# C^ (CCaret) — Syntax Documentation

## File Extension
- `.cc` for source files.
- C^ doesn't have header files. Instead, we have traits.

## Philosophy
Meaningful. Accurate. Simple. Maximum Performance.

## Upgrades
- C^ has performed a 10-20% syntax upgrade which includes some new features and deprecation of legacy C things which had mistakes.
- C^ has redesigned the default C memory model by 50-65% which gives absolute memory safety while preserving full control over the program.

## Learning
Learning C^ is about a weekend plan for a C or C++ developer.
